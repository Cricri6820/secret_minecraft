/!\ Cette secteur conserne uniquement au Minecraft Java /!\

Vous pouvez obtenir un baton de débogage avec la commande /give @p minecraft:debug_stick 1

Il vous permet de modifier les propriété d'un bloc, cela fonctionne seulement en créative

Clique droit sur le bloc: modifier la propriété selectionné d'un bloc

Shift + clique droit sur le bloc: séléctionner la propriété


Il existe aussi une monde de débogage, pour y accéder, il faut être dans le menu de création du monde => changer le type du monde en maintenant "shift" juste tant il s'écrit "débug", mais vous serez en mode spectateur
