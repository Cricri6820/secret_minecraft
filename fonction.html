GENERAL | Vous pouvez modifier le spawner en faisant shift + clique droit dessus avec un oeuf de mob

GENERAL | Vous pouvez choisir la taille de slime et de cube de magma avec /summon slime|magma_cube <coordonné> {size = <nombre>}

COMPARAISON | Les poulpes peuvent spawner dans la rivière dans la version Java, mais pas la Bedrock Edition

COMPARAISON | On peut écrire en couleur sur le panneau et dans le livre en version Bedrock, mais pas en Java

COMPARAISON | Le /summon giant (zombie géant) existe dans la version Java, mais pas dans la version Bedrock

JAVA | Le commande /gamemode 1 et /gamemode 0 a été supprimé en 1.13 et on doit utiliser /gamemode creative et /gamemode survival

JAVA | L'état de bloc a été supprimé dans les commandes en 1.13 (/give @p wool 1 2 => /give @p magenta_wool 1)

JAVA | Vous pouvez afficher la durabilité + l'ID d'un item avec F3 + G

JAVA | Dans la vieille version, le bloc Balise s'appélait "Work in progress"

BEDROCK | Il est possible d'activer le mode "Education Edition" depuis le menu du monde pour avoir accès aux blocs qui représente les élément chimique + des machines de chimie

BEDROCK | Vous pouvez accèder aux blocs qui vont arriver officiellement sur Minecraft en activant le Gameplay Exerimental dans le menu de création de monde

DIVERS | Il y a 256 stronghold (forteresse) dans la version Java, mais il y a seulement 3 dans la version Bedrock
